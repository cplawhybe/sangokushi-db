# 한위 선양 (漢魏禪讓)

## 개요
| 항목 | 내용 |
|------|------|
| 시기 | 건안 25년 = 연강 원년 = 황초 원년 (220년) 10월 |
| 장소 | 번양정(繁陽亭) 수선대(受禪臺) |
| 주체 | [[헌제]] 유협 → [[조비]] |
| 결과 | 후한(後漢) 멸망, 위(魏) 건국 |

## 정사 기록
> 출처: [정사] 삼국지 권2 「위서·문제기」 황초 원년 기록

### 경과
1. 220년 1월 [[조조]] 사망 → [[조비]] 위왕 계승
2. 같은 해 10월 [[헌제]]가 선위 조서 발포
3. 조비는 세 차례 사양 형식 거친 뒤 수락
4. 번양정 수선대에서 황제 즉위. 연호를 황초(黃初)로 변경
5. [[헌제]]를 산양공(山陽公)으로 봉함

### 헌제 처우
- 하내군(河內) 산양현(山陽) 1만 호 식읍
- 한나라 조정 연호 계속 사용 허용
- 천자 의례로 하늘에 제사 허용
- 상서 올릴 때 '신(臣)'이라 칭하지 않아도 됨
- 아들 4명 열후 봉작
✅ [정사] 본문 기록

### 제도 개편
- 상국 → 사도, 어사대부 → 사공, 봉상 → 태상, 낭중령 → 광록훈, 대리 → 정위, 대농 → 대사농으로 명칭 변경
- 환관의 각 부서 령(令) 이상 임명 금지 — 황금 간책에 새겨 보존

## 주요 인물
| 인물 | 역할 |
|------|------|
| [[조비]] | 선양 수락, 위 황제 즉위 |
| [[헌제]] | 선위 조서 발포, 산양공으로 강등 |
| 화흠(華歆) | 어사대부로서 책명 전달 |
| 왕랑(王朗) | 선양 과정 참여 |

## 동시대 반응
- **[[유비]]**: 221년 한(漢)의 정통을 계승한다며 황제 즉위 → 촉한 건국
- **[[손권]]**: 당분간 위에 복속하는 자세를 취하다 229년 독자적으로 황제 즉위

## 사료 주의사항
- 선양 절차(세 차례 사양 등)는 정치적 형식. 실질은 찬탈에 가까움.
- 문제기에는 선양을 정당화하는 참위설(讖緯說) 상주문이 다수 수록 — 위나라 측의 정치 선전 성격.
- 헌제의 실제 의사는 알 수 없음. 정사 헌제기(獻帝紀) 별도 확인 필요.
