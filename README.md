# 삼국지 마스터 DB

정사 삼국지 + 후한서 기반 대체역사 소설 데이터베이스.

## 폴더 구조

| 폴더 | 용도 |
|------|------|
| `_템플릿` | 노트 작성 템플릿 |
| `_inbox` | 빠른 메모 임시 보관 |
| `인물` | 열전 기반 인물 노트 |
| `사건` | 전투·정변·외교 사건 |
| `지명` | 주·군·현·요충지 |
| `세력` | 위·촉·오·군웅 세력 개요 |
| `사료` | 정사·후한서 원문 메모 |
| `소설작업` | 프로젝트별 집필 공간 |

## 사료 구분 원칙

노트 작성 시 출처를 반드시 명시.

- `[정사]` — 삼국지 본문 (진수)
- `[배송지주]` — 배송지 주석
- `[후한서]` — 후한서 (범엽)
- `[불명]` — 출처 불확실

## Dataview 쿼리 예시

인물 전체 목록:
```dataview
TABLE 자, 소속, 생년, 몰년
FROM "인물"
SORT 소속 ASC
```

특정 세력 인물:
```dataview
TABLE 자, 직위
FROM "인물"
WHERE 소속 = "위"
SORT 이름 ASC
```
